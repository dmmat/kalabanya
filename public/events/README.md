# Портрети гостей (необов'язкові)

Поклади сюди квадратні портрети персонажів — гра покаже їх у картці події
замість емодзі. Якщо файлу немає, лишається емодзі (нічого не ламається).

Формат: **WebP, квадрат ~512×512**, той самий мальований стиль, що й сцени.

| Файл             | Гість |
|------------------|-------|
| `bird.webp`      | горобець, що п'є з калюжі |
| `frog.webp`      | жаба Кума (з'являється і повертається) |
| `boat.webp`      | паперовий кораблик на воді |
| `truck.webp`     | важка фура над ямою |
| `snail.webp`     | равлик-крамар зі «скойкою-крамницею» |
| `heron.webp`     | сіра чапля на одній нозі |
| `cat.webp`       | чорний місячний кіт |
| `fisherman.webp` | старий дід-рибалка з вудкою |
