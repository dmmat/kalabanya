# Ілюстровані сцени калабані

Поклади сюди свої малюнки калюжі — гра автоматично почне їх показувати
замість процедурної (намальованої кодом) калюжі. Якщо файлів немає,
гра плавно відкочується до процедурного варіанту (нічого не ламається).

Очікувані імена файлів (формат `.jpg`, співвідношення ~16:9, бажано ≥1600px завширшки):

| Файл             | Коли показується                          | Що на ньому |
|------------------|-------------------------------------------|-------------|
| `day-full.jpg`   | день, вода > 66%                          | повна калюжа, віддзеркалює небо й хмари |
| `day-mid.jpg`    | день, вода 33–66%                         | калюжа підсихає, береги мулисті |
| `day-dry.jpg`    | день, вода < 33%                          | майже висохла, дрібні калюжки в ямі |
| `night-full.jpg` | ніч, вода > 50%                           | повна калюжа вночі, віддзеркалює ліхтар |
| `night-dry.jpg`  | ніч, вода < 50%                           | мала калюжка в ямі, відблиск ліхтаря |

«Ніч» вмикається у сутінках/на світанку (коли на небі видно зорі).

> Порада: експортуй ілюстрації як прогресивний JPEG ~70–80% якості,
> щоб сторінка лишалась легкою.
