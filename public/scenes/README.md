# Ілюстровані сцени калабані

Тут лежать намальовані сцени калюжі. Гра показує їх замість процедурної
(намальованої кодом) калюжі, перемикаючи залежно від рівня води та часу доби.
Якщо якогось файлу немає — гра плавно відкочується до процедурного варіанту.

| Файл              | Коли показується            | Що на ньому |
|-------------------|-----------------------------|-------------|
| `day-full.webp`   | день, вода > 66%            | повна калюжа, віддзеркалює небо й хмари |
| `day-mid.webp`    | день, вода 33–66%           | калюжа підсихає, береги мулисті |
| `day-dry.webp`    | день, вода < 33%            | майже висохла, дрібні калюжки в ямі |
| `night-full.webp` | ніч, вода > 50%             | повна калюжа вночі, віддзеркалює ліхтар |
| `night-dry.webp`  | ніч, вода < 50%             | мала калюжка в ямі, відблиск ліхтаря |

«Ніч» вмикається у сутінках/на світанку (коли на небі видно зорі).

Поточні файли оптимізовано: WebP, ширина 1600px, ≈50–100 КБ кожен.
Щоб замінити — поклади файл із тим самим іменем (бажано WebP, ~1600px завширшки).
