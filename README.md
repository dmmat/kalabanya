# 💧 КАЛАБАНЯ

**Поетична інкрементальна гра-роглайк про калюжу, що висихає.**

Ти — калабаня (діалектне українське слово для калюжі). Сонце п'є тебе краплю
за краплею. Тримайся до **сутінків** кожного дня, крути **слот-машину прогнозу
погоди** й вирішуй: ризикнути ще одним, спекотнішим днем — чи забрати
**Сутність** і піти у спокій.

🎮 **Грати:** https://dmmat.github.io/kalabanya/

## Особливості

- 🎰 **Слот неба** — прогноз погоди як слот-машина; три однакові символи дають комбо.
- 🌗 **Цикл день-ніч** — небо живе від світанку до сутінків, зорі та місяць уночі.
- 🌊 **Мрія рости** — калабаня → ставок → озеро → … → океан (ранг за об'ємом).
- 🌡️ **Глобальне потепління** — невідворотний випар, що з ~10-го дня дужчає й зрештою вбиває (рогалик: програвати — це ок).
- 🚀 **Дні Випробувань** — кожен 10-й день особливий, без прогнозу (спека, запуск ракети, засуха, курна буря, затемнення); гравця не попереджають — він сам запам'ятовує.
- 💦 **Гості та події** — друзі з пам'яттю (жаба, кіт, равлик…), хитруни (на вигляд добрі, потай шкодять), гості з таймером; події прив'язані до часу доби.
- 🍀 **Прихована Вдача** — невидимий коефіцієнт від твоїх рішень; впливає на Колесо Фортуни.
- 🎡 **Колесо Фортуни** — рідкісний івент: дуже добре / дуже лихо / нічого; можна відмовитись.
- 🌥️ **Престиж** — «Велике Випаровування»: скидання заради вічних Хмар і небесних дарів (нескінченна ескалація).
- 🏆 **Досягнення** — звичайні та **приховані** («???» до відкриття), з popup-сповіщеннями.
- 🔼 **Дві гілки прокачування** — тимчасові «поглиблення» (за воду) та вічні «дари» (за сутність).
- 📱 **PWA** — встановлюється на телефон/десктоп, працює **офлайн** (Workbox precache + кеш шрифтів), екран не гасне під час гри.
- 💾 **Збереження** — автозбереження + експорт/імпорт коду будь-коли (переносить і активний забіг).
- 🔊 **Звук** і 🖱️ **тематичний курсор** (крапля / брижі), болотяна палітра.
- 🔎 **SEO** — мета-теги, Open Graph, JSON-LD, manifest, sitemap.

## Ілюстровані сцени

Гра працює одразу з намальованою кодом калюжею. Якщо покласти власні
малюнки в `public/scenes/` (див. [`public/scenes/README.md`](public/scenes/README.md)),
вони автоматично замінять процедурну графіку, перемикаючись за рівнем води
та часом доби.

## Структура коду

Колишній моноліт `App.jsx` (~2600 рядків) розкладено на три шари, а сам
`App.jsx` (~200 рядків) — тонкий диригент:

- **`src/game/`** — чиста ігрова логіка: `balance`, `events`, `characters`,
  `festivals`, `weather`, `wheel`, `achievements`, `sky`.
- **`src/hooks/`** — стан і поведінка як React-хуки: `useGame` (корінь) +
  `useGameLoop`, `usePersistence`, `useWakeLock`.
- **`src/ui/`** — презентаційні компоненти (вівтар, оверлеї, попапи), що
  отримують усе через props.

Повна карта модулів, граф залежностей і «де що міняти» — у
[`docs/ARCHITECTURE.md`](docs/ARCHITECTURE.md).

## Розробка

Потрібен **Node.js 24+** (є `.nvmrc` — досить `nvm use`).

```bash
npm install
npm run dev      # локальний сервер розробки
npm run build    # збірка у dist/
npm run preview  # перегляд зібраного
```

## Деплой

Пуш у гілку `main` автоматично збирає й публікує гру на **GitHub Pages**
через `.github/workflows/deploy.yml`. У налаштуваннях репозиторію увімкни
**Settings → Pages → Build and deployment → Source: GitHub Actions**.

Базовий шлях задано як `/kalabanya/` у `vite.config.js`. Для іншого хоста:
`BASE=/ npm run build`.
